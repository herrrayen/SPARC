# Brand Asset Guide

Place the official event assets in this folder using these exact names:

- sparc-lightning.png
  - Source: provided lightning-based SPARC logo image.
  - Usage: source image for browser tab favicon/logo in light mode and fallback.

- sparc-lightning.svg
  - Source: SVG wrapper using `sparc-lightning.png` with tighter icon padding.
  - Usage: browser tab favicon/logo in light mode for a slightly zoomed tab icon.

- sparc-lightning-white.svg
  - Source: SVG wrapper using the updated `sparc-lightning-white.png`.
  - Usage: browser tab favicon/logo in dark mode with a slightly zoomed icon.

- sparc-logo-white.png
  - Source: provided white event logo image.
  - Usage: active header branding on dark/green backgrounds.

- sparc-bg.png
  - Source: provided social media futuristic sustainable-tech background.
  - Usage: home hero background.

CCSJ venue gallery images (for carousel on the Venue page) should be placed in this same folder using:

- ccsj-ben-arous-1.jpg
- ccsj-ben-arous-2.jpg
- ccsj-ben-arous-3.jpg
- ccsj-ben-arous-4.jpg

Notes:
- These files populate the Venue carousel automatically.
- You can replace image content anytime while keeping the same filenames.

Notes:
- Keep transparent background on the logo image if available.
- Recommended hero background minimum width: 1920px.
- Current code has fallback styling if files are missing, but final visual identity appears once these files are added.

Venue-specific assets are stored in `public/assets/venue/`:

- ccsj-logo.jpg
  - Usage: venue logo block on the Venue page.

- ccsj-image.jpg
  - Usage: venue preview image on the Venue page.
